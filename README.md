# MatchChallenge
Testing different methods for pattern matching to find longest repeating substring that an input string may consist of.

## Current methods
- Substring+SpanSlice
- Split+Replace
- Regex
- Linq
- Pure Regex
- Offset+Modulo
- Kmp
    - Heap alloc
    - Stack alloc
    - Array pool

## Testing tool
- BenchmarkDotNet
